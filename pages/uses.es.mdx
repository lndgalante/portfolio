# Usos

## Configuración

Podes encontrar todas mis configuraciones en este [repositorio](https://github.com/lndgalante/tools-configuration).

## Stack Tecnológico

- Deploy: Vercel
- Librería UI: React.js
- Lenguaje: TypeScript
- React Framework: Next.js
- Estilos: Chakra UI or Tailwind
- Manejo de estados: Jotai / React Query
- Testing: Jest / React Testing Library / Cypress

## Herramientas Principales

- Navegador: Brave
- Terminal: iTerm
- Editor: VSCode
- Diseño: Figma
- Administración: Notion
- Cliente REST/GraphQL: Insomnia

## Herramientas Secundarias

- Responsiveness: ResponsivelyApp
- Documentación: Dash
- Utilidades: DevUtils
