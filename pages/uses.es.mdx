# Usos

## Configuración

Podes encontrar todas mis configuraciones en este [repositorio](https://github.com/lndgalante/tools-configuration).

## Stack Tecnológico

### Frontend

- Deploy: Vercel
- Lenguaje: TypeScript
- Librería UI: React.js
- React Framework: Next.js
- Estilos: Chakra UI
- Animaciones: Framer Motion
- Manejo de estados: Zustand / Jotai / React Query
- Testing: Jest / React Testing Library / Cypress

### Backend

- TypeScript Runtime: Deno
- Base de datos: PostgreSQL / Supabase
