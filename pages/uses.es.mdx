# Usos

## Tech Stack

- Deployment: Vercel
- UI Library: React.js
- Language: TypeScript
- React Framework: Next.js
- Server State: React Query
- Styling: Chakra UI or Tailwind
- Testing: Jest / React Testing Library / Cypress

## Primary Tools

- Browser: Brave
- Terminal: iTerm
- Editor: VSCode
- Design: Figma
- REST Client: Insomnia
- Management: Notion

- Utilities: DevUtils
